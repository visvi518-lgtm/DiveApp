# DiveAppSwiftPlayground

This is a lightweight Swift Playgrounds version of DiveApp.

## Included

- Login UI with mock state changes
- Profile setup mock flow
- Main tab UI
- Home
- Dive log list, detail, and in-memory add form
- Training mock screen
- Community list and detail mock
- My page and certificate mock

## Excluded

- Real Naver/Google social login
- Backend API calls
- Access token refresh
- Keychain storage
- Database sync
- Image upload
- Push notifications
- Admin features

## iPad Run

1. Copy the `DiveAppSwiftPlayground` folder to iPad.
2. If Swift Playgrounds does not open it directly, rename the folder to `DiveAppSwiftPlayground.swiftpm`.
3. Open it in Swift Playgrounds.
4. Run the app.

The app uses only SwiftUI and in-memory mock data.
